# 日记 (Journal)

本目录用于存放每日记录、周复盘、月度总结等时间序列笔记。

## 笔记类型

### 1. 每日记录
- **文件名**：`YYYY-MM-DD.md`
- **内容**：当日工作要点、思考、收获、待办
- **频率**：每天工作结束时

### 2. 周复盘
- **文件名**：`周复盘-YYYY-MM-DD.md`
- **模板**：使用`05-Templates/周复盘模板.md`
- **频率**：每周五下午
- **内容**：本周成果、挑战、学习、下周计划

### 3. 月度总结
- **文件名**：`月度总结-YYYY-MM.md`
- **频率**：每月最后一天
- **内容**：月度目标完成情况、关键成果、改进方向

### 4. 季度回顾
- **文件名**：`季度回顾-YYYY-QQ.md`
- **频率**：每季度末
- **内容**：季度目标回顾、团队表现、战略调整

## 使用建议

### 1. 保持简洁
- 每日记录不超过10分钟
- 重点记录关键决策、学习收获
- 避免写成流水账

### 2. 定期回顾
- **每周**：查看本周记录，准备周复盘
- **每月**：回顾每日记录，提炼模式
- **每季度**：分析趋势，调整工作方式

### 3. 关联链接
- 链接相关项目：[[项目名称]]
- 链接会议记录：[[会议记录-YYYYMMDD]]
- 链接学习笔记：[[学习主题]]

### 4. 隐私考虑
- 敏感信息可加密或使用代号
- 个人反思可标记为私有
- 团队相关记录注意保密

## 价值体现

1. **记忆外置**：减少脑力负担，专注思考
2. **模式识别**：通过回顾发现工作模式和效率瓶颈
3. **成长轨迹**：记录学习和成长过程
4. **决策依据**：为未来决策提供历史参考
5. **情绪管理**：通过书写整理思绪，管理情绪
