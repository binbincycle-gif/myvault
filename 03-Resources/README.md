# 资源知识库 (Resources)

本目录用于存放技术栈、行业知识、AI学习等参考资料。

## 子目录说明

### 技术栈
- **.NET**：C#、ASP.NET Core、Entity Framework等
- **React**：React Hooks、状态管理、性能优化等
- **K8s微服务**：容器化、服务网格、监控告警等

### 行业知识
- **服饰零售**：行业趋势、业务流程、客户痛点
- **CRM领域**：产品功能、竞品分析、最佳实践
- **SaaS趋势**：商业模式、定价策略、客户成功

### AI学习
- **Agent技术**：AI代理原理、架构设计、应用场景
- **Prompt工程**：提示词技巧、最佳实践、模板设计
- **工具评测**：Claude、GPT、开源工具对比评测

### 效率工具
- **知识管理**：Obsidian、Notion等知识管理工具
- **开发工具**：VS Code、Git、Docker等开发效率工具
- **团队协作**：飞书、钉钉、Jira等团队协作工具
- **其他工具**：各类提升个人和团队效率的工具

## 使用建议

1. **知识结构化**：将学习内容按主题分类整理
2. **持续更新**：定期回顾和更新知识库内容
3. **关联实践**：与`01-Projects/`中的项目实践结合
4. **团队共享**：优质资源可分享给团队成员
