# 知识库索引

本文档用于维护本知识库的索引和导航信息。

## 📁 目录结构说明

| 目录              | 用途    | 使用说明                    |
| --------------- | ----- | ----------------------- |
| `00-Inbox/`     | 收件箱   | 临时存放未处理的信息、想法、待办事项      |
| `01-Projects/`  | 项目    | 当前进行中的项目文档，使用[[项目笔记模板]] |
| `02-Areas/`     | 责任领域  | 长期关注的工作领域，如技术中心管理、团队建设等 |
| `03-Resources/` | 资源知识库 | 技术栈、行业知识、AI学习等参考资料      |
| `04-Archives/`  | 归档    | 已完成的项目、过时的资料            |
| `05-Templates/` | 模板库   | 各种笔记模板，方便快速创建结构化内容      |
| `06-Journal/`   | 日记    | 每日记录、周复盘                |
| `07-Meta/`      | 元知识   | 关于知识库本身的文档（如本索引）        |
| `Clippings/`    | 剪藏    | 收藏的文章、资料（保留原目录）         |
| `Excalidraw/`   | 图表    | 架构图、流程图（保留原目录）          |
| `aboutme/`      | 个人介绍  | 个人信息（保留原目录）             |
| `copilot/`      | AI对话  | Claude Copilot相关（保留原目录） |
| `代码片段库/`        | 代码片段  | 代码记录（保留原目录）             |

## 🔄 工作流程

### 每日流程
1. **早晨检视**：查看`00-Inbox/`和`06-Journal/YYYY-MM-DD.md`
2. **工作记录**：会议、决策、代码片段实时记录
3. **晚间复盘**：更新日记，清理收件箱

### 每周流程
1. **周一规划**：创建本周复盘文档
2. **周五复盘**：填写周复盘模板，归档完成工作

### 剪藏处理
1. **收集** → `00-Inbox/`或`Clippings/`
2. **处理** → 使用Copilot的`Summarize`命令
3. **关联** → 添加相关链接
4. **应用** → 转化为`03-Resources/`中的知识

## 🤖 AI工具集成

### 自定义Copilot指令
- `知识关联生成`：为新笔记推荐相关链接
- `周报生成`：基于周复盘生成上级汇报
- `Summarize`：文章摘要（已有）

## 📊 数据看板（可选）

如需使用数据看板，需安装Obsidian Dataview插件。然后在笔记中使用以下查询：

```dataview
TABLE status, priority, start_date, end_date
FROM "01-Projects"
WHERE status != "completed"
SORT priority DESC, start_date
```

## 🚀 开始使用

1. **模板使用**：在`05-Templates/`中选择合适的模板创建新笔记
2. **收件箱管理**：每天清空`00-Inbox/`，将信息分类到相应目录
3. **标签应用**：为每篇笔记添加合适的标签，便于检索
4. **链接构建**：使用`[[笔记标题]]`建立笔记间的关联
