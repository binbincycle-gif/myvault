# 责任领域 (Areas)

本目录用于存放长期关注的工作领域和职责相关的文档。

## 子目录说明

### 技术中心管理
- 团队管理、流程优化、技术规划
- 绩效考核、人才培养、文化建设
- 预算管理、资源分配、风险管理

### 团队建设
- 招聘计划、面试记录、入职培训
- 团队活动、文化建设、员工关怀
- 技能培训、知识分享、职业发展

### AI战略规划
- AI技术趋势研究、工具评估
- AI产品规划、落地场景设计
- AI团队建设、技能培训计划

### 个人发展
- 职业规划、学习计划、技能提升
- 反思总结、成就记录、成长轨迹

## 使用建议

1. **按领域分类**：将相关文档放入对应子目录
2. **定期回顾**：每个季度回顾各领域进展
3. **目标对齐**：与公司战略和个人发展目标对齐
4. **知识沉淀**：将经验转化为可复用的方法论
